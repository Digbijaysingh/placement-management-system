## Placement Management System

## 📌 Overview
A web-based Placement Management System built using Python Django to manage students, companies, recruiters, job openings, and placement activities.

## ✨ Features
- Student Registration
- Company Management
- Recruiter Management
- Job Posting
- Placement Sessions
- Admin Dashboard
- User Authentication

## 🛠️ Tech Stack
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

## 🚀 Installation

```bash
git clone https://github.com/Digbijaysingh/placement-management-system.git
cd placement-management-system
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
