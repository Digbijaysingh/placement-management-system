# Placement Management System

A full-stack **Placement Management System** developed using **Python**, **Django**, **Django REST Framework (DRF)**, **HTML**, **CSS**, **JavaScript**, and **SQLite**. The application streamlines campus recruitment by providing dedicated portals for students and recruiters, enabling placement session management, job posting, application tracking, secure REST APIs with JWT Authentication, and interactive API documentation using Swagger.

---

## Features

- Student Registration & Authentication
- Recruiter Registration & Authorization
- Company Management
- Placement Session Management
- Job Posting & Management
- Job Application Tracking
- Resume Upload & Management
- Role-Based Access Control
- REST APIs using Django REST Framework (DRF)
- JWT Authentication
- Swagger (OpenAPI) Documentation
- Responsive User Interface

## Installation

```bash
git clone https://github.com/Digbijaysingh/placement-management-system.git

cd placement-management-system

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
